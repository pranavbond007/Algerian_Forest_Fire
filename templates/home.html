About the Algerian Forest Fires Dataset

This dataset contains meteorological data from the Algerian forests (Bejaia and Sidi Bel-Abbes) for predicting Fire Weather Index (FWI) and identifying fire occurrences.

Feature Descriptions:

  • Temperature (¬∞C): Daily average temperature.
  • RH (%): Relative Humidity.
  • WS (km/h): Wind Speed.
  • Rain (mm): Rainfall amount.
  • FFMC: Fine Fuel Moisture Code ‚Äì moisture content of surface litter.
  • DMC: Duff Moisture Code ‚Äì moisture content in decomposed organic matter.
  • ISI: Initial Spread Index ‚Äì rate of fire spread.
  • Classes: 0 = No Fire, 1 = Fire.
  • Region: 1 = Bejaia Region, 2 = Sidi Bel-Abbes Region.
